# Simple Interest Calculator

A simple command-line application written in Bash shell scripting to calculate simple interest based on user inputs.

## Description
This project is designed to help users compute simple interest quickly by providing key inputs through the terminal.

## Features
* Accepts Principal amount ($P$)
* Accepts Annual rate of interest ($R$)
* Accepts Time period in years ($T$)
* Outputs the calculated Simple Interest and Total Amount
